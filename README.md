# FinWise 💰
A personal finance expert chatbot

## Features
- Multi-turn conversation with memory
- Covers budgeting, investing, debt, and retirement
- Clean dark UI with suggested prompts

## Tech Stack
- Vanilla HTML/CSS/JS
- Anthropic Claude API

## Setup
1. Clone the repo
2. Open index.html in your browser
3. Add your Anthropic API key
